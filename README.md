# Exploreuk

Service scripts for ExploreUK.

## Installation

This is not published on RubyGems.

Build and install:

    $ git clone https://github.com/uklibraries/exploreuk_services.git
    $ cd exploreuk_services
    $ bundle
    $ rake install

Add this line to your application's Gemfile:

```ruby
gem 'exploreuk'
```

And then execute:

    $ bundle
